# Python - Almost a Circle

This project implements Base, Rectangle, and Square classes
with serialization, file I/O, and full unittest coverage.

## Files

- `models/base.py` - Base class with JSON serialization
- `models/rectangle.py` - Rectangle class inheriting Base
- `models/square.py` - Square class inheriting Rectangle
- `tests/test_models.py` - Unittest for all classes

## Run Tests

```bash
python3 -m unittest tests/test_models.py -v
```
